# InovCorp CRM

InovCorp CRM is a modern, full-featured Customer Relationship Management (CRM) web application designed to help businesses manage leads, entities, people, deals, and activities efficiently.

## Tech Stack
- **Backend:** Laravel 12 (PHP)
- **Frontend:** Vue 3, Inertia.js, Pinia, ShadcnVue
- **Database:** MySQL (or compatible)

## Features
- Entity and People management with advanced validation
- Deal and activity tracking
- Customizable fields and statuses
- Role-based access and policies
- Responsive UI with modern design system
- API-driven architecture for easy integration

## Purpose
This project aims to provide a robust, extensible CRM platform for small and medium businesses, following best practices in software architecture (SOLID, service layer, DTOs, resources, etc.).
